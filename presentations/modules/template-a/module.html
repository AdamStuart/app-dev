Background

You write apps for a variety of reasons.
  • It's a Google Summer of Code, or CS Class Assignment
  • You need to import niche file or data formats for network analysis
  • You have data with a higher dimensionality that requires custom processing
  • You want to optimize access to a particular database

Setup

You need a variety of tools and techniques to write an app:
  • Assuming Eclipse Environment for Compiling
  • Git seems to work best from the command line
  • Mvn installation and introduction

Establish your starting point

  • Add a menu item or button to launch a URL: sample1
  • Implement a simple layout algorithm: polyPath
  • Implement a complex layout algorithm: EntOptLayout
  • Import an XML file: Wikipathways
  • Support Network Search Bar and add a results dialog: StringApp
  • Change Menu Bar Configuration

The Development Cycle for Cytoscape apps

  • CyActivator registers the commands you are adding
  • EBIRR: Edit / Build / Install / Run / Repeat
  • Tasks are the unit of work. Each has a 'factory' that attaches to menu items and buttons.
